1. Введение
Метод Зейделя является одним из основных итерационных методов решения систем линейных уравнений. В отличие от других методов, он позволяет эффективно решать большие системы, занимая меньше памяти
    В данной работе реализованы две версии алгоритма:
Последовательная (SEQ) — обычная последовательная версия
Параллельная (MPI) — версия с распределением между несколькими процессами
    Цель работы: реализовать параллельный алгоритм и исследовать его производительность в сравнении с последовательной версией

2. Постановка задачи
Необходимо решить систему линейных уравнений ax = b методом Зейделя
    Входные данные:
using InType = std::vector<double>;
    Выходные данные:
using OutType = std::vector<double>;
    Формат данных:
На вход подается размер системы, точность и максимальное число итераций, а на выходе мы получаем вектор решения
    Ограничения:
Размер системы должен быть больше 0, точность должна быть больше 0 и число итераций должно быть больше 0, Матрица a должна обеспечивать сходимость

3. Базовый алгоритм (последовательный)
1) Инициализация данных: проверяем что размер системы N > 0 если не так, то выдается ошибка. Инициализируется атрица A размером N × N
2) Итерационный процесс Зейделя: Для каждого i от 0 до N-1 - сохраняется строе значение, вычисляется новое значение, вычисление максимального изменения, когда достигается лимит итераий выходим из цикла 
3) Проверка точности: если residual_norm < точность * 1000
    Сложность:
Временная: O(N²)

4. Схема параллелизации
Распределение данных: матрица A распределяется по строкам между процессами затем каждый процесс обрабатывает свой блок строк, а векторы b и x синхронизируются на всех процессах
    Для каждой итерации:
1. Каждый процесс вычисляет обновления для своих строк
2. MPI_Allgather собирает обновления x_i со всех процессов
3. Процесс 0 проверяет сходимость
4. MPI_Bcast рассылает решение о продолжении или остановке
    Роли процессов:
Rank 0: управляет сходимостью и отвечает за итоговый результат. Все остальные процессы занимаются вычислениями
